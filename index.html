02 — Categories

Темы хакатона

Каждый выбирает одну категорию. Темы объявляются в вечер вторника. Пятая тема — «Фристайл» — требует согласования с жюри.

01 / КАТЕГОРИЯ💡

Новая фича

Принципиально новая функция для ОСИНТ-расследований: новый способ сбора, обогащения или визуализации данных об объекте. Инновация как главная ценность.

02 / КАТЕГОРИЯ🔧

Вспомогательный инструмент

ИИ-ассистент, который помогает аналитику в рутинных задачах — поиск, агрегация, первичная обработка данных об объектах.

03 / КАТЕГОРИЯ⚡

Упрощение работы

Автоматизация ручных этапов: верификация данных, построение связей, подготовка отчётов — быстрее и точнее, чем вручную.

04 / КАТЕГОРИЯ📊

Работа с данными

ИИ-решение для анализа больших массивов ОСИНТ-данных: графы связей, аномалии, паттерны поведения, временные ряды активности.

05 / КАТЕГОРИЯ · FREESTYLE🎲

Фристайл

Свободная тема: собственная ОСИНТ-идея, которая не вписывается в категории 01–04. Главное — польза для расследовательской работы.

⚠ Тему необходимо заранее согласовать с одним из членов жюри

06 — Rules

Правила

Прочитайте внимательно перед участием. Незнание правил не освобождает от ответственности.

Участники

  • Хакатон индивидуальный — каждый сам за себя
  • Один участник — один проект
  • Регистрация до вечера вторника
  • Совместная разработка не допускается

Технические требования

  • Весь код — в предоставленном репозитории
  • README обязателен с инструкцией запуска
  • Проект должен запускаться на презентации
  • Open-source библиотеки и AI API — разрешены

Презентация

  • 5–7 минут + 3–5 минут на вопросы жюри
  • Живая демонстрация продукта обязательна
  • Слайды — опционально
  • Порядок — жеребьёвка в пятницу утром

Ограничения

  • Чужой код — только с указанием источника
  • Все разработки остаются собственностью компании
  • Несоответствие теме — снятие с конкурса
  • Решения жюри окончательны
